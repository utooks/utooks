# 우뚝이들(UTOOKs) 프로젝트
우뚝이들(UTOOKs)은 개발자들이 함께하여 실제 서비스로 만들어내는 실무형 모임입니다.

- 모임명: 우뚝이들
- 영문명: UTOOKs
- 현재 구성인원: 5명

## 우리가 모인 이유
개발자는 혼자 빠르게 갈 수 있지만, 함께하면 더 멀리 갈 수 있다고 믿습니다.  
우뚝이들은 기술 공유, 협업 경험, 실제 서비스 개발을 하나의 흐름으로 연결해 "**결과물이 남는 성장**"을 만드는 것을 목표로 합니다.

## 구성원
- 웹 프론트엔드 개발자
- 백엔드 개발자
- SAP/데이터 모델러
- 데이터 엔지니어
- 인프라 엔지니어

## 핵심 활동
- 프로젝트 주기: 3~6개월 단위 MVP 개발
- 문제 정의 → 핵심기능 선정 → 팀 구성 → 기획/설계(UI/UX, 아키텍처) → 개발 → 테스트/피드백 → 개선 및 릴리스

## 참여 원칙
- 프로젝트 참여는 자율
- 문서화, 회고, 기술 공유는 권장
- 직군/경력/관점을 존중하며 소통
- 일회성 이벤트보다 루틴 중심 운영

## 활동 공간
- 오프라인: 용산 아지트
- 온라인: GitHub 중심 협업(문서, 이슈, 코드)

## 이런 분과 함께하고 싶습니다
- 아이디어는 있지만, 서비스를 어떻게 만들지 고민인 분
- 사이드 프로젝트를 끝까지 만들고, 결과물을 남기고 싶은 분
- 서로의 성장을 돕는 커뮤니티 문화를 지향하는 분
